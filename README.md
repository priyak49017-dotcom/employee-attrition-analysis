# Employee Attrition & Performance Analysis

## 1. Introduction
This project analyzes employee attrition using Python to identify trends based on department, salary, experience, and performance.

## 2. Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## 3. Dataset Description
The dataset contains employee details such as department, experience, salary, performance rating, and attrition status.

## 4. Data Cleaning
Missing values in experience and salary were handled using median and mean methods.

## 5. Feature Engineering
Attrition was converted into binary format and experience levels were created.

## 6. Analysis & Visualization
Department-wise attrition and salary trends were analyzed using bar charts.

## 7. Key Insights
- IT department shows higher retention
- Low performance employees show higher attrition
- Senior employees have lower attrition
